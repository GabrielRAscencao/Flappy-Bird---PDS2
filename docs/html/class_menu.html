Descrição detalhada

Gerencia e exibe a tela de menu principal do jogo.

Esta classe é responsável por mostrar as opções do menu (Jogar, Ranking, Configurações, Sair), permitir a entrada de apelido para o jogador e lidar com a navegação e seleção de opções.
