Descrição detalhada

Gerencia e exibe a tela de "Game Over" do jogo.

Esta classe é responsável por mostrar a pontuação do jogador, os recordes (pessoal e geral) e oferecer opções para reiniciar o jogo ou voltar ao menu principal.
